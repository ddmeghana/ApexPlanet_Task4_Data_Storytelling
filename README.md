# Superstore Business Performance Analysis – Task 4

This repository contains Task 4 of my Data Analytics Internship at ApexPlanet, focused on Data Storytelling and Statistical Validation using the Superstore Sales dataset.

## Objective
The objective of this task is to synthesize insights from previous analysis into a business narrative and validate a key finding using hypothesis testing.

## Tools Used
- Python
- Pandas
- SciPy
- Jupyter Notebook
- PowerPoint
- Tableau

## Hypothesis Tested
High discounts reduce profitability.

## Statistical Test Used
Independent two-sample t-test was used to compare profits between low-discount and high-discount orders.

## Key Business Findings
- Technology is the most profitable category.
- Consumer segment contributes the highest order share.
- West region shows strong sales performance.
- High discounts negatively impact profitability.
- Profit margin should be monitored along with sales.

## Deliverables
- Final presentation deck
- Hypothesis testing notebook
- Hypothesis testing summary

## Conclusion
The analysis highlights that profitability is strongly influenced by discount strategy. Businesses should optimize discounts and focus on profitable categories and regions.
